o que significa linguagem de programação

Uma linguagem de programação é um vocabulário e um conjunto de regras gramaticais para instrução de um computador ou dispositivo de computação a executar tarefas específicas.

O termo linguagem de programação geralmente se refere a linguagens de alto nível, como BASIC, C, C ++, COBOL, Java, FORTRAN, Ada e Pascal.
